# T1110 – Brute Force

## Description
This technique simulates brute-force authentication attempts, commonly used to obtain unauthorized access to accounts.

## Command Used
Invoke-AtomicTest T1110.001

## Logs Observed
- Failed login attempts (Event ID 4625)
- Possible successful login (Event ID 4624)

## Analysis
Repeated failed login attempts followed by a successful login may indicate a successful brute-force attack.

## Detection Opportunities
- Monitor repeated authentication failures
- Detect login attempts from the same IP
- Identify successful logins after multiple failures

## MITRE ATT&CK Mapping
- Tactic: Credential Access
- Technique: T1110
